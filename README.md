# Gesture Controlled Mouse 🎯

This project uses computer vision to control mouse using hand gestures.

## 🚀 Features
- Move mouse with index finger
- Click using pinch gesture
- Smooth cursor movement

## 🛠️ Tech Used
- Python
- OpenCV
- MediaPipe
- PyAutoGUI

## ▶️ How to Run

```bash
pip install opencv-python mediapipe pyautogui
python main.py
